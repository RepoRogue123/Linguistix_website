Model Architecture & Performance

Overview

Our speaker recognition system uses a Neural Network architecture specialized for audio feature classification. The model is trained on extracted features from audio samples, rather than raw audio, making it efficient and accurate.

Architecture Details

Neural Network Architecture
  • Input Layer: 49 neurons (LDA features)
  • Hidden Layer 1: 128 neurons with Sigmoid activation
  • Hidden Layer 2: 128 neurons with Sigmoid activation
  • Output Layer: 50 neurons (one per speaker)
Training Details
  • Optimizer: Adam
  • Loss Function: Cross Entropy Loss
  • Learning Rate: 0.001
  • Batch Size: 32
  • Epochs: 50

Feature Extraction

Our system extracts 17 audio features from each voice sample:

  • 13 Mel-frequency cepstral coefficients (MFCCs)
  • Spectral centroid
  • Spectral bandwidth
  • Spectral rolloff
  • Zero crossing rate

These features capture key characteristics of a person's voice, creating a unique "fingerprint" that allows for accurate identification.

Limitations & Future Work

While our system achieves high accuracy, it has some limitations:

  • Limited performance in very noisy environments
  • Requires at least 3 seconds of speech for reliable identification
  • Currently limited to identifying 50 different speakers

Future work will focus on:

  • Implementing noise reduction preprocessing
  • Expanding the model to identify more speakers
  • Reducing the required audio length for identification
